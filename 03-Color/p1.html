DEFINING COLOR VALUES

Color in CSS most often specifies in the following formats:

  1. Color Keyword-like red,green,blue,orange and etc
  2. Hexadecimal (2^8 =256) value - Like #0000000,#ffffff and etc
  3. RGB (RED,GREEN,BLUE) Value -Like rgb(255,0,0),rgb(255,255,255),rgb(0,0,0) and etc
  4. RGBA (RED,GREEN,BLUE,ALPHA) value - like rgb(255,0,0,0.5),rgb(255,255,255,0.5),rgba(0,0,0,0.5) and etc
  5. HSL (HUE,SATURATION,LIGHTNESS) value
  6. HSLA (HUE,SATURATION,LIGHTNESS,ALPHA) value
